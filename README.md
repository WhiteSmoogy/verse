# verse-rs

A small Rust interpreter for a Verse-aligned language subset.

This is not a full implementation of Epic Games Verse yet. It is a practical
compiler/interpreter foundation: lexer, parser, AST, evaluator, CLI, REPL, tests,
and examples. New language work should track documented Verse syntax instead of
adding project-specific syntax.

## Current language

```verse
factorial(n:int):int = if (n <= 1) {
    1
} else {
    n * factorial(n - 1)
}

print(factorial(5))
```

## Usage

```powershell
cargo run -- examples/factorial.verse
cargo run -- examples/functions.verse
cargo run -- examples/enums.verse
cargo run -- examples/structs.verse
cargo run -- examples/classes.verse
cargo run -- examples/loops.verse
cargo run -- examples/arrays.verse
cargo run -- examples/maps.verse
cargo run -- examples/tuples.verse
cargo run -- examples/options.verse
cargo run -- examples/returns.verse
cargo run -- check examples/factorial.verse
cargo run -- ast examples/factorial.verse
cargo run
```

The REPL keeps bindings between inputs:

```text
verse> x:int = 41
41
verse> x + 1
42
```
